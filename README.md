
# 🏦 Django Bank Management System

A simple Bank Management System built using Django that performs core banking operations like account creation, deposits, withdrawals, and fund transfers.

## 🚀 Features

* 👤 Customer Management (Create, Update, Delete)
* 💳 Account Management
* 💰 Deposit & Withdraw Money
* 🔄 Fund Transfer Between Accounts
* 📜 Transaction History Tracking
* ✅ Basic Validations (Insufficient balance, active account, etc.)

## 🛠️ Tech Stack

* Python
* Django
* SQLite (default)

## 📂 Modules

* Customer
* Account
* Transactions

## ⚙️ Setup

```bash
git clone https://github.com/KUNTALinGITHUB/django-bank-management-system.git
cd django-bank-management-system
pip install -r requirements.txt
python manage.py runserver
```

## 🎯 Purpose

This project is built for practicing Django CRUD operations along with real-world business logic implementation.
